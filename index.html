Chemistry Tool

New! Chemistry Tool now does atoms and polyatomic ions. Enter something like phosphate or NO3- to find its name/formula. Enter something like He, 2 or Helium to find more details on that element,

Using this tool, you can convert between color, wavelength, energy, and frequency. If you enter something that Chemistry Tool doesn't understand, it'll take you to the Wikipedia page on whatever you typed.
Please enter your input without units. The program will recognize your units for you.
If you wish to input frequency or energy, please input it in scientific notation with an e. Example: 5e14 or 3e-19

Useful Symbols:

λ = wavelength
c = speed of light
v or f = frequency
h = Planck constant

Useful Equations

λ = 1/v (wavelength = 1/frequency)
ΔE = -2.178*10e-18 J * (1/nf^2 * 1/nI^2) (Bohr equation)
